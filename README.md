# Murshidak Project

Murshidak is a smart student guidance application designed to help students manage and apply for their subjects, track availability, and interact with a built-in chatbot for guidance. This project is developed as a graduation project and demonstrates full-stack development with a modern architecture.

---

## Features

- **Subject Management:** View and apply for subjects with available slots.
- **Chatbot Integration:** Get instant guidance and answers related to your studies.
- **User Profile:** Manage your profile, including personal info and settings.
- **Notifications:** Stay updated about important events and subject deadlines.
- **Multi-Layer Architecture:** Clean separation of responsibilities for maintainability.

---

## Tech Stack

- **Backend:** ASP.NET Core Web API
- **Frontend:** (HTML,CSS,JS,React)
- **Database:** (SQL Server)
- **Authentication:** JWT-based authentication
- **Mapping:** AutoMapper
